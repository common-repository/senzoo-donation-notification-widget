== Description ==

**We no longer support this plugin.**

Senzoo Widget makes it simple to add donation form with notification widget which has PayPal, Amazon FPS and Google Checkout on your WordPress blog.

Related Links:

* <a href="http://senzoo.net/">Senzoo Homepage</a>

== Installation ==

1. Upload `senzoo-widget` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Settings -> Senzoo Widget and start entering your info.

== Frequently Asked Questions ==
